{%- comment -%} Sticky site header. Sits at top:0 with z-30 so page content scrolls underneath. Two responsive states: - At lg+ (>= 1024 px): logo on the left, nav items inline on the right. - Below lg: logo on the left, hamburger button on the right. Tapping the hamburger toggles `mobileOpen` (Alpine.js data), which expands a stacked vertical menu below the header. Nav items come from `site.nav` in _config.yml. To add a page to the nav, add an entry there — no edits to this file required. The banner image (sustainable-fsa-banner.svg) has its own internal padding, which is why the wrapper uses tight `py-0.5` vertical padding rather than the more typical py-3. {%- endcomment -%}
